Tips for developing for mobile devices

  1. Don't try to replicate the main page for mobile devices
  2. Don't use large images that take a long time to load.
  3. Try to avoid javascript, php etc
  4. Stick with plain HTML5 whenever possible.
  5. Don't use tables and instead use one long scrolling site
  6. Add meta tag
  7. Use CSS whenever possible
